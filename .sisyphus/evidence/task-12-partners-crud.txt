# Task 12: Partners CRUD Mutations - COMPLETED

## Summary
Added CRUD mutations to convex/partners.ts for managing partners data.

## Mutations Added

### 1. createPartner
- Creates a new partner with all required fields
- Returns the new partner ID
- Fields: name, logo_url, type, url (optional), order, quote (optional), industry (optional), icon (optional)

### 2. updatePartner  
- Updates an existing partner by ID
- Accepts optional fields for partial updates
- Filters out undefined values before patching
- Returns the updated partner document

### 3. deletePartner
- Deletes a partner by ID

### 4. reorderPartners
- Batch updates order field for multiple partners
- Accepts array of { id, order } objects
- Updates each partner's order in sequence

## Schema Fields (partners table)
- name: string
- logo_url: string
- type: string ("certification", "client", "tech_partner")
- url: optional string
- order: number
- quote: optional string
- industry: optional string
- icon: optional string
