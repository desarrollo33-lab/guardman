## Files Modified

## Features Implemented

### HeroesList.tsx
- Display all heroes with page_slug
- Search by title, page_slug, subtitle
- Filter by is_active status (All/Active/Inactive)
- CRUD buttons (Create, Edit, Delete)
- Toggle active status inline
- Background type indicator (YouTube/Image)
- CTAs and Badges count display
- Loading and empty states

### HeroForm.tsx
- **Page selector dropdown**: home, servicios, nosotros, contacto, blog, cotizar
- **Title field** (required)
- **Subtitle field** (optional)
- **Background type toggle**: YouTube vs Image
  - When YouTube: youtube_id input with preview link
  - When Image: image_url and mobile_image_url inputs with preview
- **CTAs array editor**:
  - Add/remove CTAs
  - Edit text, href, variant (primary/secondary/outline)
- **Badges array editor**:
  - Add/remove badges
  - Edit text and icon (shield-check, check-circle, star, award, clock, users, building)
- **is_active toggle**
- Modal with form validation

## Convex Functions Used
- `api.heroes.getAllHeroes` - List all heroes
- `api.heroes.createHero` - Create new hero
- `api.heroes.updateHero` - Update existing hero
- `api.heroes.deleteHero` - Delete hero
